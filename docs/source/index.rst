Introduction
===========

The Modular Backtesting System is a flexible and extensible framework for backtesting trading strategies. It provides a modular architecture that allows users to easily implement and test their own trading strategies, data sources, and portfolio management techniques.

Features
--------

* Modular architecture for easy extension and customization
* Event-driven design for realistic simulation
* Support for multiple data sources
* Flexible strategy implementation
* Comprehensive performance metrics
* Visualization tools for analysis
* Command-line interface for easy execution
* Configuration-based setup
